### 소인수분해 | 함정 2개

#### 🟡 중상 ⭐⭐ | 약수의 개수 조건 역추적
**함정:** 약수의 개수가 주어지고 원래 수를 찾는 역방향 문제

**해결 핵심**
* ✓ 약수 개수 = (지수+1)들의 곱을 역이용
* ✓ 여러 가지 경우의 수 고려 (예: 12 = 12, 6×2, 4×3, 3×2×2)
* ✓ 가장 작은 자연수를 찾을 때는 작은 소수에 큰 지수 배치

**예시:** 약수의 개수가 12개인 가장 작은 자연수는?

---

#### 🟡 중상 ⭐⭐ | 최대공약수/최소공배수 조건 문제
**함정:** 두 수의 최대공약수와 최소공배수가 주어지고 두 수를 찾는 문제

**해결 핵심**
* ✓ (최대공약수) × (최소공배수) = 두 수의 곱
* ✓ 두 수를 최대공약수로 나눈 몫은 서로소
* ✓ 조건을 만족하는 모든 쌍 찾기

**예시:** 최대공약수가 6, 최소공배수가 72인 두 자연수의 순서쌍은?

### 정수와 유리수 | 함정 2개

**함정:** 여러 수의 곱/나눗셈에서 결과의 부호를 판정하는 문제

**해결 핵심**
* ✓ 음수의 개수만 세기 (홀수면 음수, 짝수면 양수)
* ✓ 0이 포함되면 결과는 무조건 0
* ✓ 절댓값의 크기는 부호 결정 후 계산

**예시:** a < 0 < b < c일 때, a²bc³ / ab²c의 부호는?

#### 🟡 중상 ⭐⭐ | 절댓값 조건 문제
**함정:** 절댓값이 포함된 등식/부등식에서 경우 나누기

**해결 핵심**
* ✓ |a| = a (a≥0), |a| = -a (a<0)
* ✓ |a-b|의 해석: a와 b 사이의 거리
* ✓ a, b의 부호에 따라 경우 분류

**예시:** |a| = 3, |b| = 5, ab < 0일 때, a - b의 값은?

**예시:** a = -3일 때, -a², (-a)², -(-a)²의 값을 각각 구하시오.

#### 🟡 중상 ⭐⭐ | 식의 값 변형 문제
**함정:** 직접 대입이 어려워 식을 변형해야 하는 문제

**해결 핵심**
* ✓ a + b, ab의 값이 주어지면 대칭식 활용
* ✓ 분수식은 분자, 분모를 따로 계산
* ✓ 조건식을 변형하여 구하는 식에 대입

**예시:** x + 1/x = 3일 때, x² + 1/x²의 값은?

### 일차방정식 | 함정 2개

#### 🟡 중상 ⭐⭐⭐ | 계수에 문자가 포함된 방정식
**함정:** ax = b 형태에서 a = 0인 경우를 놓침

**해결 핵심**
* ✓ a ≠ 0이면 x = b/a
* ✓ a = 0, b = 0이면 해가 무수히 많음
* ✓ a = 0, b ≠ 0이면 해가 없음

**예시:** (k-2)x = k + 4가 해가 없으려면 k의 값은?

#### 🟡 중상 ⭐⭐ | 활용 문제의 조건 검증
**함정:** 구한 해가 문제 상황에서 불가능한 경우

**해결 핵심**
* ✓ 나이, 개수 등은 자연수 조건
* ✓ 거리, 속력, 시간은 양수 조건
* ✓ 문제의 맥락에서 답의 타당성 검토

**예시:** 연속하는 세 자연수의 합이 24일 때, 세 수를 구하시오.

### 좌표와 그래프 | 함정 2개

#### 🟡 중상 ⭐⭐ | 대칭점 좌표
**함정:** x축, y축, 원점 대칭에서 부호 변경 혼동

**해결 핵심**
* ✓ x축 대칭: (a, b) → (a, -b) [y부호 변경]
* ✓ y축 대칭: (a, b) → (-a, b) [x부호 변경]
* ✓ 원점 대칭: (a, b) → (-a, -b) [둘 다 변경]

**예시:** 점 A(-3, 5)의 x축 대칭점을 B, B의 y축 대칭점을 C라 할 때, C의 좌표는?

---

#### 🟡 중상 ⭐⭐ | 사분면 판정
**함정:** 부등식 조건에서 점이 위치하는 사분면 결정

**해결 핵심**
* ✓ 제1사분면: x > 0, y > 0
* ✓ ab > 0이면 같은 부호, ab < 0이면 다른 부호
* ✓ 축 위의 점은 사분면에 속하지 않음

**예시:** ab < 0, a + b > 0일 때, 점 (a, b)는 몇 사분면?

### 정비례와 반비례 | 함정 2개

#### 🟡 중상 ⭐⭐ | 비례상수 부호와 그래프
**함정:** 비례상수 a의 부호에 따른 그래프 위치 혼동

**해결 핵심**
* ✓ y = ax: a > 0이면 1, 3사분면, a < 0이면 2, 4사분면
* ✓ y = a/x: a > 0이면 1, 3사분면, a < 0이면 2, 4사분면
* ✓ 그래프가 지나는 점으로 부호 결정

**예시:** y = a/x의 그래프가 점 (-2, 3)을 지날 때, 이 그래프가 지나는 사분면은?

---

#### 🟡 중상 ⭐⭐ | 정비례/반비례 판별
**함정:** 식의 형태만 보고 판별하면 틀리는 경우

**해결 핵심**
* ✓ y = ax (a ≠ 0)만 정비례
* ✓ y = a/x (a ≠ 0)만 반비례
* ✓ y = 2x + 1, y = x²는 정비례/반비례 아님

**예시:** xy = 6을 만족하는 x, y의 관계를 설명하시오.

## 중1-2학기

### 기본 도형 | 함정 2개

#### 🟡 중상 ⭐⭐⭐ | 동위각과 엇각 혼동
**함정:** 평행선이 아닌 상황에서 동위각/엇각이 같다고 가정

**해결 핵심**
* ✓ 동위각 = 엇각은 "평행선일 때만" 성립
* ✓ 평행하지 않으면 동위각 ≠ 엇각
* ✓ 그림을 그려 위치 확인 필수

**예시:** l // m이 아닐 때, 동위각의 크기가 다른 것을 보이시오.

---

#### 🟡 중상 ⭐⭐ | 맞꼭지각과 동위각 구분
**함정:** 여러 직선이 교차할 때 각의 관계 혼동

**해결 핵심**
* ✓ 맞꼭지각: 항상 같음 (평행 여부 무관)
* ✓ 동위각/엇각: 평행선일 때만 같음
* ✓ 각 위치 먼저 파악 후 관계 적용

**예시:** 세 직선이 한 점에서 만날 때, 맞꼭지각을 모두 찾으시오.

### 작도와 합동 | 함정 2개

#### 🟠 상 ⭐⭐⭐ | SSA는 합동 조건이 아님
**함정:** 두 변과 그 끼인각이 아닌 각이 같을 때 합동으로 착각

**해결 핵심**
* ✓ SSS, SAS, ASA만 합동 조건
* ✓ SSA(두 변과 끼인각이 아닌 각)는 합동 조건 아님
* ✓ AAA도 합동 조건 아님 (닮음 조건)

**예시:** AB = DE, BC = EF, ∠C = ∠F일 때, △ABC ≅ △DEF인가?

---

#### 🟡 중상 ⭐⭐ | 대응 순서 맞추기
**함정:** 합동 기호에서 대응하는 꼭짓점 순서를 틀림

**해결 핵심**
* ✓ △ABC ≅ △DEF이면 A↔D, B↔E, C↔F
* ✓ 대응변: AB=DE, BC=EF, CA=FD
* ✓ 대응각: ∠A=∠D, ∠B=∠E, ∠C=∠F

**예시:** △ABC ≅ △DEF에서 BC에 대응하는 변은?

---

### 평면도형의 성질 | 함정 2개

#### 🟠 상 ⭐⭐⭐ | 정다각형의 한 내각/외각 계산
**함정:** 내각과 외각 공식을 혼동하거나 n값 대입 실수

**해결 핵심**
* ✓ 한 외각 = 360°/n
* ✓ 한 내각 = 180° - (한 외각) = 180°(n-2)/n
* ✓ 내각의 합 = 180°×(n-2)

**예시:** 정다각형의 한 내각이 140°일 때, 변의 개수는?

---

#### 🟡 중상 ⭐⭐ | 부채꼴 넓이와 호의 길이 관계
**함정:** 중심각이 다른 부채꼴의 넓이 비교

**해결 핵심**
* ✓ 호의 길이 l = 2πr × (θ/360°)
* ✓ 넓이 S = πr² × (θ/360°) = (1/2)rl
* ✓ 반지름이 다르면 중심각만으로 비교 불가

**예시:** 반지름 6, 중심각 60°인 부채꼴의 넓이는?

---

### 입체도형의 성질 | 함정 2개

#### 🟡 중상 ⭐⭐ | 회전체의 전개도
**함정:** 원뿔의 전개도에서 부채꼴 중심각 계산

**해결 핵심**
* ✓ 원뿔 옆면 부채꼴의 호의 길이 = 밑면 원의 둘레
* ✓ 2πl × (θ/360°) = 2πr (l: 모선, r: 밑면 반지름)
* ✓ θ = 360° × (r/l)

**예시:** 밑면 반지름 3, 모선 길이 9인 원뿔의 전개도에서 부채꼴 중심각은?

---

#### 🟡 중상 ⭐⭐ | 정다면체의 면, 모서리, 꼭짓점
**함정:** 정다면체 종류별 특성 혼동

**해결 핵심**
* ✓ 정다면체는 5가지뿐 (정사, 정육, 정팔, 정십이, 정이십)
* ✓ 오일러 공식: V - E + F = 2
* ✓ 각 정다면체의 면의 모양과 개수 암기

**예시:** 정십이면체의 꼭짓점과 모서리의 개수는?

### 자료의 정리와 해석 | 함정 2개

#### 🟡 중상 ⭐⭐ | 계급값과 평균 계산
**함정:** 도수분포표에서 평균 계산 시 계급값 사용

**해결 핵심**
* ✓ 계급값 = (계급의 양 끝값의 합)/2
* ✓ 평균 = Σ(계급값 × 도수) / 전체 도수
* ✓ 원본 자료의 평균과 다를 수 있음

**예시:** 도수분포표에서 (계급값 × 도수)의 합이 450, 전체 도수가 30일 때 평균은?

#### 🟡 중상 ⭐⭐ | 상대도수의 해석

**해결 핵심**
* ✓ 상대도수가 같아도 전체 도수가 다르면 도수 다름
* ✓ 상대도수의 합 = 1
* ✓ 두 집단 비교 시 상대도수 사용

**예시:** A반(30명)과 B반(40명)에서 같은 계급의 상대도수가 0.2일 때, 각 반의 도수는?

## 중2-1학기

### 유리수와 순환소수 | 함정 2개

#### 🟠 상 ⭐⭐⭐ | 유한소수 판별
**함정:** 기약분수로 만들지 않고 분모만 보고 판별

**해결 핵심**
* ✓ 반드시 기약분수로 약분 후 판별
* ✓ 기약분수 분모의 소인수가 2, 5뿐이면 유한소수
* ✓ 분모에 2, 5 외 소인수가 있으면 무한소수(순환소수)

**예시:** 6/15는 유한소수인가?

---

#### 🟠 상 ⭐⭐ | 순환소수를 분수로 변환
**함정:** 순환마디 파악 실수 또는 변환 공식 오류

**해결 핵심**
* ✓ 0.abc̅ = (abc - ab)/900 형태 파악
* ✓ 순환마디 자릿수만큼 9, 비순환 자릿수만큼 0
* ✓ 변환 후 기약분수로 정리

**예시:** 0.1̅2̅를 기약분수로 나타내시오.

### 식의 계산 (지수법칙) | 함정 2개

#### 🟠 상 ⭐⭐⭐ | 지수법칙 혼동
**함정:** aᵐ × aⁿ = aᵐⁿ으로 착각, (aᵐ)ⁿ = aᵐ⁺ⁿ으로 착각

**해결 핵심**
* ✓ aᵐ × aⁿ = aᵐ⁺ⁿ (같은 밑의 곱 → 지수 덧셈)
* ✓ (aᵐ)ⁿ = aᵐⁿ (거듭제곱의 거듭제곱 → 지수 곱셈)
* ✓ (ab)ⁿ = aⁿbⁿ

**예시:** (2³)² × 2⁴ ÷ 2⁵의 값은?

---

#### 🟡 중상 ⭐⭐ | 동류항 판별
**함정:** 문자는 같지만 차수가 다른 항을 동류항으로 착각

**해결 핵심**
* ✓ 동류항: 문자와 차수가 모두 같은 항
* ✓ 3x²y와 5xy²는 동류항 아님
* ✓ 상수항끼리는 동류항

**예시:** 3a²b, -5ab², 7a²b 중 동류항끼리 묶으시오.

---

### 일차부등식 | 함정 2개

#### 🟠 상 ⭐⭐⭐ | 음수 곱셈 시 부등호 방향
**함정:** 양변에 음수를 곱하거나 나눌 때 부등호 방향 유지

**해결 핵심**
* ✓ 양수 곱/나눗셈 → 부등호 방향 유지
* ✓ 음수 곱/나눗셈 → 부등호 방향 반대로
* ✓ -x > 3의 양변에 -1을 곱하면 x < -3

**예시:** -2x + 6 ≥ 0을 풀면?

---

#### 🟡 중상 ⭐⭐ | 해의 범위 표현
**함정:** 경계값 포함 여부(≤, <)와 수직선 표시(●, ○) 혼동

**해결 핵심**
* ✓ ≤, ≥: 경계값 포함 → ● (채운 점)
* ✓ <, >: 경계값 미포함 → ○ (빈 점)
* ✓ "미만/초과"와 "이하/이상" 구분

**예시:** x > 3을 수직선에 나타내시오.

---

### 연립일차방정식 | 함정 2개

#### 🟠 상 ⭐⭐⭐ | 해가 무수히 많음/해 없음
**함정:** 일반적인 연립방정식처럼 풀다가 특수한 경우를 놓침

**해결 핵심**
* ✓ 두 식이 같아지면 → 해가 무수히 많음
* ✓ 0 = k (k≠0) 형태면 → 해가 없음
* ✓ 계수 비교: a₁/a₂ = b₁/b₂ = c₁/c₂면 무수히 많음

**예시:** x + 2y = 3, 2x + 4y = 6의 해는?

---

#### 🟠 상 ⭐⭐ | 세 방정식에서 공통해
**함정:** 세 방정식을 모두 만족하는 해 찾기

**해결 핵심**

## 중3-1학기

### 제곱근과 실수 | 함정 2개

#### 🔴 최상 ⭐⭐⭐ | √a²의 값
**함정:** √a² = a로 단순히 생각 (a가 음수일 때 오류)

**해결 핵심**
* ✓ √a² = |a| (절댓값)
* ✓ a ≥ 0이면 √a² = a
* ✓ a < 0이면 √a² = -a

**예시:** √(-5)²의 값은?

---

#### 🟠 상 ⭐⭐⭐ | √a와 a의 제곱근 구분
**함정:** √4 = ±2로 착각

**해결 핵심**
* ✓ √4 = 2 (양의 제곱근만)
* ✓ 4의 제곱근 = ±2 (양, 음 모두)
* ✓ √ 기호 자체가 양의 제곱근을 의미

**예시:** √9의 값과 9의 제곱근을 각각 구하시오.

---

### 근호를 포함한 식의 계산 | 함정 2개

#### 🔴 최상 ⭐⭐⭐ | 덧셈/뺄셈 착각
**함정:** √a + √b = √(a+b)로 계산

**해결 핵심**
* ✓ √a + √b ≠ √(a+b) (성립 안 함!)
* ✓ √a × √b = √(ab) (이것만 성립)
* ✓ 덧셈/뺄셈은 근호 안이 같을 때만 가능

**예시:** √2 + √3의 값을 간단히 하시오. (답: 간단히 안 됨)

---

#### 🟠 상 ⭐⭐⭐ | 분모의 유리화 (켤레식)
**함정:** 분모가 √a ± √b일 때 유리화 방법

**해결 핵심**
* ✓ 분모가 √a이면 √a를 곱함
* ✓ 분모가 √a + √b이면 √a - √b를 곱함 (켤레식)
* ✓ (√a + √b)(√a - √b) = a - b

**예시:** 1/(√3 + √2)를 유리화하시오.

---

### 다항식의 곱셈 (곱셈 공식) | 함정 2개

#### 🔴 최상 ⭐⭐⭐ | 중간항 부호 실수
**함정:** (a-b)² = a² - b²로 착각 (중간항 누락)

**해결 핵심**
* ✓ (a+b)² = a² + 2ab + b²
* ✓ (a-b)² = a² - 2ab + b² (중간항 -2ab)
* ✓ (a+b)(a-b) = a² - b² (이건 중간항 없음)

**예시:** (3x - 2)²을 전개하시오.

---

#### 🟠 상 ⭐⭐ | 곱셈 공식 변형 활용
**함정:** a² + b²를 인수분해하려다 막힘

**해결 핵심**
* ✓ a² + b² = (a+b)² - 2ab
* ✓ a² + b² = (a-b)² + 2ab
* ✓ (a-b)² = (a+b)² - 4ab

**예시:** a + b = 5, ab = 3일 때, a² + b²의 값은?

### 인수분해 | 함정 2개

#### 🔴 최상 ⭐⭐⭐ | 완전제곱식 vs 합차공식 혼동
**함정:** a² - b²을 (a-b)²으로 인수분해

**해결 핵심**
* ✓ a² - b² = (a+b)(a-b) ← 합차공식
* ✓ a² - 2ab + b² = (a-b)² ← 완전제곱식
* ✓ "제곱 - 제곱"과 "제곱 - 2배 + 제곱" 구분

**예시:** x² - 9를 인수분해하시오.

#### 🟠 상 ⭐⭐⭐ | 공통인수 완전 추출

**해결 핵심**
* ✓ 모든 항의 공통인수를 먼저 완전히 묶기
* ✓ 묶고 남은 식도 인수분해 가능한지 확인
* ✓ 인수분해 결과를 전개하여 검산

**예시:** 2x² - 8을 완전히 인수분해하시오.

## 중3-2학기

### 삼각비 | 함정 2개

#### 🔴 최상 ⭐⭐⭐ | sin, cos, tan 정의 혼동
**함정:** 대변, 인접변, 빗변을 혼동하여 삼각비 오계산

**해결 핵심**
* ✓ sin θ = 대변/빗변 (opposite/hypotenuse)
* ✓ cos θ = 인접변/빗변 (adjacent/hypotenuse)
* ✓ tan θ = 대변/인접변 (opposite/adjacent)
* ✓ "소대빗, 코인빗, 탄대인"으로 암기

**예시:** 직각삼각형에서 빗변 5, 대변 3일 때 sin θ는?

---

#### 🟠 상 ⭐⭐⭐ | 특수각 값 혼동
**함정:** 30°, 45°, 60°의 삼각비 값을 서로 바꿔 기억

**해결 핵심**
* ✓ sin: 30° → 1/2, 45° → √2/2, 60° → √3/2
* ✓ cos: 30° → √3/2, 45° → √2/2, 60° → 1/2
* ✓ sin과 cos은 30°, 60°에서 서로 바뀜

**예시:** sin 60° + cos 30°의 값은?

---

### 삼각비의 활용 | 함정 2개

#### 🟠 상 ⭐⭐ | 직각삼각형이 없는 경우
**함정:** 일반 삼각형에 직접 삼각비를 적용하려 함

**해결 핵심**
* ✓ 직각삼각형이 없으면 수선(높이)을 그어 만들기
* ✓ 삼각형 넓이 = (1/2) × 밑변 × 높이
* ✓ 높이 = (한 변) × sin(그 변의 대각)

**예시:** 두 변이 5, 8이고 끼인각이 60°인 삼각형의 넓이는?

---

#### 🟡 중상 ⭐⭐ | 넓이 공식 계수 누락
**함정:** 삼각형 넓이 공식에서 1/2를 빠뜨림

**해결 핵심**

#### 🟡 중상 ⭐⭐ | 다항식 나눗셈에서 빠진 차수

### 나머지정리와 인수정리 | 함정 2개

#### 🔴 최상 ⭐⭐⭐ | 나누는 식의 부호
**함정:** (x-a)로 나눈 나머지가 P(-a)라고 착각

**해결 핵심**
* ✓ (x-a)로 나눈 나머지 = P(a)
* ✓ (x+a)로 나눈 나머지 = P(-a)
* ✓ 조립제법: x-2로 나눌 때 2 사용, x+3으로 나눌 때 -3 사용

**예시:** P(x) = x³ - 2x + 1을 x + 1로 나눈 나머지는?

---

#### 🔴 최상 ⭐⭐⭐ | 이차식으로 나눈 나머지
**함정:** (x-a)(x-b)로 나눈 나머지가 상수라고 착각

**해결 핵심**
* ✓ (일차식)으로 나눈 나머지 = 상수
* ✓ (이차식)으로 나눈 나머지 = 일차식 또는 상수
* ✓ R(x) = px + q로 놓고 P(a), P(b)를 이용해 p, q 결정

**예시:** P(x)를 (x-1)(x-2)로 나눈 나머지를 구하시오. (단, P(1) = 3, P(2) = 7)

### 인수분해 (고등) | 함정 2개

#### 🔴 최상 ⭐⭐⭐ | 인수정리로 인수 찾기
**함정:** P(a) = 0이면 (x+a)가 인수라고 착각

**해결 핵심**
* ✓ P(a) = 0이면 (x-a)가 인수
* ✓ P(-a) = 0이면 (x+a)가 인수
* ✓ 상수항의 약수를 대입해 인수 찾기

**예시:** x³ - 6x² + 11x - 6을 인수분해하시오.

---

#### 🟠 상 ⭐⭐ | 복이차식 인수분해
**함정:** x⁴ + ax² + b 형태를 직접 인수분해하려 함

**해결 핵심**
* ✓ x² = t로 치환하여 t에 대한 이차식으로 변환
* ✓ t² + at + b를 인수분해 후 t를 x²로 복원
* ✓ x²의 인수분해도 확인 (x² - 1 = (x+1)(x-1))

**예시:** x⁴ - 5x² + 4를 인수분해하시오.

### 복소수 | 함정 2개

#### 🟠 상 ⭐⭐⭐ | i의 거듭제곱

**해결 핵심**
* ✓ i¹ = i, i² = -1, i³ = -i, i⁴ = 1 (4개 주기)

* ✓ i⁴⁵ = i^(44+1) = (i⁴)¹¹ × i = i

#### 🟠 상 ⭐⭐ | 복소수 나눗셈

**해결 핵심**

* ✓ 분모, 분자에 분모의 켤레복소수를 곱함
* ✓ (a + bi)(a - bi) = a² + b² (실수)

### 이차방정식의 판별식 | 함정 2개

**해결 핵심**
* ✓ "서로 다른 두 실근" → D > 0
* ✓ "실근을 가진다" (중근 포함) → D ≥ 0
* ✓ "적어도 하나의 실근" → D ≥ 0

**해결 핵심**
* ✓ ax² + bx + c = 0의 두 근 α, β에 대해
* ✓ α + β = -b/a (부호 주의!)
* ✓ αβ = c/a

**예시:** x² - 5x + 6 = 0의 두 근의 합과 곱은?

#### 🔴 최상 ⭐⭐⭐ | 대칭식 계산

**해결 핵심**
* ✓ α² + β² = (α + β)² - 2αβ
* ✓ α³ + β³ = (α + β)³ - 3αβ(α + β)
* ✓ 1/α + 1/β = (α + β)/αβ

**예시:** 두 근의 합이 4, 곱이 2일 때, 두 근의 제곱의 합은?

**해결 핵심**
* ✓ 모든 x에서 y > 0: a > 0이고 D < 0
* ✓ 모든 x에서 y < 0: a < 0이고 D < 0
* ✓ 그래프를 그려서 확인

**예시:** y = x² - 2x + k가 항상 양수이려면 k의 범위는?

* ✓ 중선을 2:1로 내분

**예시:** (x+1)² + (y-3)² = 16의 중심과 반지름은?

---

#### 🟠 상 ⭐⭐⭐ | 일반형에서 원 조건
**함정:** 일반형이 항상 원을 나타낸다고 착각

**해결 핵심**
* ✓ x² + y² + Ax + By + C = 0에서
* ✓ (A/2)² + (B/2)² - C > 0이어야 원
* ✓ = 0이면 점, < 0이면 공집합

**예시:** x² + y² - 2x + 4y + 5 = 0은 원인가?

---

### 원과 직선 | 함정 2개

#### 🔴 최상 ⭐⭐⭐ | 원과 직선의 위치 관계
**함정:** d와 r 비교에서 등호 포함 여부

**해결 핵심**
* ✓ d < r: 두 점에서 만남
* ✓ d = r: 접함 (한 점)
* ✓ d > r: 만나지 않음

**예시:** 원 x² + y² = 9와 직선 y = x + k가 접하려면 k는?

---

#### 🔴 최상 ⭐⭐⭐ | 원의 접선 (원 밖의 점)
**함정:** 원 밖의 점에서 접선의 개수와 접선의 방정식

**해결 핵심**
* ✓ 원 밖의 점에서 접선은 2개
* ✓ y - y₁ = m(x - x₁)로 놓고 판별식 D = 0 이용
* ✓ 또는 접점을 (a, b)로 놓고 원 위의 점 조건 + 수직 조건

**예시:** 점 (4, 0)에서 원 x² + y² = 4에 그은 접선의 방정식은?

---

### 도형의 이동 | 함정 2개

#### 🟠 상 ⭐⭐ | 점과 방정식의 이동 부호
**함정:** 점의 이동과 방정식의 이동에서 부호 처리를 같게 함

**해결 핵심**
* ✓ 점 (x, y) → (x+a, y+b)로 평행이동
* ✓ 방정식에서는 x 대신 x-a, y 대신 y-b 대입
* ✓ 점과 방정식의 부호가 반대

**예시:** y = x²을 x축 방향으로 2, y축 방향으로 3만큼 평행이동하면?

---

#### 🟡 중상 ⭐⭐ | 대칭이동
**함정:** x축, y축, 원점 대칭의 규칙 혼동

**해결 핵심**
* ✓ x축 대칭: y → -y
* ✓ y축 대칭: x → -x
* ✓ 원점 대칭: x → -x, y → -y

**예시:** y = x² - 2x를 원점에 대해 대칭이동하면?

---

### 집합과 명제 | 함정 2개

#### 🟠 상 ⭐⭐ | 드모르간 법칙
**함정:** (A∪B)ᶜ를 Aᶜ∪Bᶜ로 착각

**해결 핵심**
* ✓ (A∪B)ᶜ = Aᶜ ∩ Bᶜ
* ✓ (A∩B)ᶜ = Aᶜ ∪ Bᶜ
* ✓ 여집합을 취하면 ∪와 ∩가 바뀜

**예시:** U = {1,2,3,4,5}, A = {1,2}, B = {2,3}일 때, (A∪B)ᶜ는?

---

#### 🔴 최상 ⭐⭐⭐ | 필요조건과 충분조건
**함정:** p → q에서 p가 필요조건이라고 착각

**해결 핵심**
* ✓ p → q가 참일 때: p는 충분조건, q는 필요조건
* ✓ "p이면 q이다"에서 p가 충분, q가 필요
* ✓ 집합으로: P ⊂ Q이면 P가 충분, Q가 필요

**예시:** "x = 2이면 x² = 4이다"에서 각각 무슨 조건인가?

---

### 절대부등식 | 함정 2개

#### 🔴 최상 ⭐⭐⭐ | 산술-기하 평균 등호 조건
**함정:** 등호 성립 조건 확인 없이 최솟값 결정

**해결 핵심**
* ✓ 산술평균 ≥ 기하평균, 등호는 a = b일 때
* ✓ 문제의 조건에서 a = b가 가능한지 확인
* ✓ 불가능하면 등호 성립 안 함 → 최솟값 아님

**예시:** x > 0일 때, x + 4/x의 최솟값은? (등호 성립: x = 2)

---

#### 🟠 상 ⭐⭐ | 양수 조건 미확인
**함정:** 음수에 산술-기하 평균을 적용

**해결 핵심**
* ✓ a, b가 양수일 때만 (a+b)/2 ≥ √(ab)
* ✓ 음수가 포함되면 적용 불가
* ✓ 조건을 확인하고 양수임을 보이기

**예시:** a > -1일 때, (a+1) + 1/(a+1)의 최솟값은? (a+1 > 0 확인)

---

### 함수 | 함정 2개

#### 🟠 상 ⭐⭐⭐ | 합성함수 순서
**함정:** (f∘g)(x)를 g(f(x))로 계산

**해결 핵심**
* ✓ (f∘g)(x) = f(g(x))
* ✓ 안쪽 함수 g를 먼저 계산
* ✓ (f∘g) ≠ (g∘f) 일반적으로

**예시:** f(x) = 2x, g(x) = x + 1일 때, (f∘g)(3)은?

---

#### 🔴 최상 ⭐⭐⭐ | 역함수 존재 조건
**함정:** 모든 함수에 역함수가 있다고 착각

**해결 핵심**
* ✓ 역함수 존재 ⟺ 일대일대응
* ✓ y = x²은 역함수 없음 (x ≥ 0으로 제한하면 있음)
* ✓ 그래프: y = x에 대해 대칭

**예시:** f(x) = x³의 역함수는?

---

### 유리함수와 무리함수 | 함정 2개

#### 🟠 상 ⭐⭐⭐ | 점근선
**함정:** y = k/(x-p) + q의 점근선을 x = -p, y = -q로 착각

**해결 핵심**
* ✓ y = k/(x-p) + q의 점근선: x = p, y = q
* ✓ 괄호 안 부호와 반대로
* ✓ 그래프는 점근선에 가까워지지만 만나지 않음

**예시:** y = 2/(x-1) + 3의 점근선은?

---

#### 🟠 상 ⭐⭐ | 무리함수의 정의역
**함정:** 근호 안이 음수가 될 수 있다고 착각

**해결 핵심**
* ✓ √(ax + b)에서 ax + b ≥ 0
* ✓ 정의역: x ≥ -b/a (a > 0일 때)
* ✓ 치역도 제한됨 (대개 y ≥ c 형태)

**예시:** y = √(2x - 4)의 정의역과 치역은?

# 고등학교 (대수 / 구 수학Ⅰ)

## 대수 (고2)

### 지수 | 함정 2개

#### 🟠 상 ⭐⭐⭐ | 유리수 지수 변환
**함정:** a^(m/n)에서 n의 위치 혼동

**해결 핵심**
* ✓ a^(m/n) = ⁿ√(aᵐ) = (ⁿ√a)ᵐ
* ✓ 분모 n이 근호의 지수
* ✓ 분자 m이 거듭제곱의 지수

**예시:** 8^(2/3)의 값은?

---

#### 🟡 중상 ⭐⭐ | a⁰ = 1
**함정:** a⁰ = 0으로 착각

**해결 핵심**
* ✓ a⁰ = 1 (a ≠ 0)
* ✓ a⁻ⁿ = 1/aⁿ
* ✓ 0⁰은 정의되지 않음

**예시:** 5⁰ + 2⁻³의 값은?

---

### 로그 | 함정 2개

#### 🔴 최상 ⭐⭐⭐ | 로그 덧셈 착각
**함정:** log(a+b) = log a + log b로 착각

**해결 핵심**
* ✓ log(ab) = log a + log b ← 곱의 로그
* ✓ log(a+b) ≠ log a + log b
* ✓ log(a/b) = log a - log b

**예시:** log 2 + log 3의 값은?

---

#### 🔴 최상 ⭐⭐⭐ | 진수 조건과 밑 조건
**함정:** 진수나 밑의 조건을 확인하지 않음

**해결 핵심**
* ✓ 진수 > 0
* ✓ 밑 > 0이고 밑 ≠ 1
* ✓ 방정식/부등식 풀이 후 조건 확인 필수

**예시:** log₂(x-1) = 3의 해는?

---

### 지수함수 | 함정 2개

#### 🟠 상 ⭐⭐⭐ | 밑에 따른 증가/감소
**함정:** a > 1일 때와 0 < a < 1일 때 혼동

**해결 핵심**
* ✓ a > 1: 증가함수 (x↑ → y↑)
* ✓ 0 < a < 1: 감소함수 (x↑ → y↓)
* ✓ 지수부등식에서 밑에 따라 부등호 방향 주의

**예시:** (1/2)ˣ > 8을 풀면?

---

#### 🟡 중상 ⭐⭐ | 지수함수의 점근선
**함정:** y = aˣ이 x축과 만난다고 착각

**해결 핵심**
* ✓ x축(y = 0)이 점근선
* ✓ 그래프는 x축에 가까워지지만 만나지 않음
* ✓ aˣ > 0 (항상 양수)

**예시:** y = 2ˣ의 그래프가 x축과 만나는 점은?

---

### 로그함수 | 함정 2개

#### 🟠 상 ⭐⭐⭐ | 밑에 따른 증가/감소
**함정:** 밑이 1보다 작을 때 부등호 방향

**해결 핵심**
* ✓ a > 1: 증가함수
* ✓ 0 < a < 1: 감소함수
* ✓ 로그부등식에서 밑에 따라 부등호 방향 바뀜

**예시:** log₀.₅ x > 2를 풀면?

---

#### 🟡 중상 ⭐⭐ | 로그함수의 정의역
**함정:** 모든 실수에서 정의된다고 착각

**해결 핵심**
* ✓ y = logₐ x의 정의역: x > 0
* ✓ y축(x = 0)이 점근선
* ✓ 항상 점 (1, 0)을 지남

**예시:** y = log₂(x-1)의 정의역은?

---

### 삼각함수 | 함정 2개

#### 🔴 최상 ⭐⭐⭐ | 사분면별 부호
**함정:** 각 사분면에서 삼각함수 부호 혼동

**해결 핵심**
* ✓ 1사분면: 모두 +
* ✓ 2사분면: sin만 +
* ✓ 3사분면: tan만 +
* ✓ 4사분면: cos만 +

**예시:** sin 150°, cos 150°, tan 150°의 부호는?

---

#### 🔴 최상 ⭐⭐⭐ | 삼각함수 기본 관계
**함정:** sin θ + cos θ = 1로 착각

**해결 핵심**
* ✓ sin²θ + cos²θ = 1 (제곱의 합!)
* ✓ tan θ = sin θ / cos θ
* ✓ 1 + tan²θ = sec²θ

**예시:** sin θ = 3/5일 때, cos θ는? (θ는 제1사분면)

---

### 삼각함수의 그래프 | 함정 2개

#### 🔴 최상 ⭐⭐⭐ | 주기 계산
**함정:** y = sin(bx)의 주기를 2πb로 착각

**해결 핵심**
* ✓ y = sin(bx)의 주기 = 2π/|b|
* ✓ y = tan(bx)의 주기 = π/|b|
* ✓ b가 클수록 주기가 짧아짐

**예시:** y = sin(2x)의 주기는?

---

#### 🟠 상 ⭐⭐ | 그래프의 이동
**함정:** y = sin(x - π/4)의 이동 방향

**해결 핵심**
* ✓ y = sin(x - a)는 오른쪽으로 a만큼 이동
* ✓ y = sin(x + a)는 왼쪽으로 a만큼 이동
* ✓ 괄호 안 부호와 반대 방향

**예시:** y = sin x를 오른쪽으로 π/3만큼 이동하면?

---

### 삼각함수의 활용 | 함정 2개

#### 🟠 상 ⭐⭐⭐ | 사인법칙과 코사인법칙 선택
**함정:** 어떤 법칙을 써야 할지 판단 못함

**해결 핵심**
* ✓ 사인법칙: 변과 대각 관계, 외접원 반지름
* ✓ 코사인법칙: 세 변 또는 두 변과 끼인각
* ✓ 주어진 조건에 맞는 법칙 선택

**예시:** 삼각형에서 두 변이 5, 7이고 끼인각이 60°일 때, 나머지 변은?

---

#### 🟠 상 ⭐⭐ | 코사인법칙 부호
**함정:** a² = b² + c² - 2bc cos A에서 마지막 항 부호

**해결 핵심**
* ✓ a² = b² + c² - 2bc cos A (빼기!)
* ✓ cos A = (b² + c² - a²) / 2bc
* ✓ A가 둔각이면 cos A < 0

**예시:** 세 변이 3, 5, 7인 삼각형의 가장 큰 각의 cos값은?

---

### 등차수열 | 함정 2개

#### 🟠 상 ⭐⭐ | 일반항 공식
**함정:** aₙ = a₁ + nd로 착각

**해결 핵심**
* ✓ aₙ = a₁ + (n-1)d
* ✓ n번째 항까지 공차 d가 (n-1)번 더해짐
* ✓ 등차중항: b = (a + c)/2

**예시:** 첫째항 3, 공차 2인 등차수열의 10번째 항은?

---

#### 🟡 중상 ⭐⭐ | 등차수열의 합
**함정:** Sₙ = n(n-1)d/2로 착각

**해결 핵심**
* ✓ Sₙ = n(a₁ + aₙ)/2
* ✓ Sₙ = n{2a₁ + (n-1)d}/2
* ✓ 첫째항과 끝항을 알면 첫 번째 공식이 편리

**예시:** 1 + 2 + 3 + ... + 100의 값은?

---

### 등비수열 | 함정 2개

#### 🔴 최상 ⭐⭐⭐ | 일반항 지수
**함정:** aₙ = a₁ × rⁿ으로 착각

**해결 핵심**
* ✓ aₙ = a₁ × r^(n-1)
* ✓ n번째 항까지 공비 r을 (n-1)번 곱함
* ✓ 등비중항: b² = ac (b = ±√ac)

**예시:** 첫째항 2, 공비 3인 등비수열의 5번째 항은?

---

#### 🔴 최상 ⭐⭐⭐ | r = 1인 경우
**함정:** 등비수열 합 공식에서 r = 1인 경우 처리

**해결 핵심**
* ✓ r ≠ 1: Sₙ = a₁(1-rⁿ)/(1-r) = a₁(rⁿ-1)/(r-1)
* ✓ r = 1: Sₙ = na₁
* ✓ r = 1이면 등차수열(공차 0)

**예시:** 2 + 2 + 2 + ... + 2 (n개)의 합은?

---

### 수열의 합 | 함정 2개

#### 🔴 최상 ⭐⭐⭐ | Σ 공식
**함정:** Σk² = n(n+1)/2로 착각

**해결 핵심**
* ✓ Σk = n(n+1)/2
* ✓ Σk² = n(n+1)(2n+1)/6
* ✓ Σk³ = {n(n+1)/2}²

**예시:** 1² + 2² + 3² + ... + 10²의 값은?

---

#### 🟠 상 ⭐⭐ | 부분분수 분해
**함정:** 소거 패턴 파악 실패

**해결 핵심**
* ✓ 1/k(k+1) = 1/k - 1/(k+1) 형태로 분해
* ✓ 연속된 항이 소거됨
* ✓ 처음 몇 개와 마지막 몇 개만 남음

**예시:** Σ(k=1 to n) 1/k(k+1)의 값은?

---

### 수학적 귀납법 | 함정 2개

#### 🟠 상 ⭐⭐ | 귀납 가정 미사용
**함정:** n = k+1 증명에서 n = k 가정을 사용하지 않음

**해결 핵심**
* ✓ 반드시 n = k일 때 성립한다는 가정을 이용
* ✓ 가정을 이용하지 않으면 귀납법이 아님
* ✓ P(k) → P(k+1)을 보여야 함

**예시:** 1 + 2 + ... + n = n(n+1)/2를 수학적 귀납법으로 증명하시오.

---

#### 🟡 중상 ⭐⭐ | n = 1 확인 생략
**함정:** n = 1일 때 성립 확인을 빠뜨림

**해결 핵심**
* ✓ 귀납법 첫 단계: n = 1(또는 시작값)에서 성립 확인
* ✓ 이 단계가 없으면 증명 불완전
* ✓ 반례가 존재할 수 있음

**예시:** n = 2부터 시작하는 명제는 n = 2에서 확인

# 고등학교 (미적분Ⅰ / 구 수학Ⅱ)

---

## 미적분Ⅰ (고2)

### 함수의 극한 | 함정 2개

#### 🔴 최상 ⭐⭐⭐ | 0/0 부정형
**함정:** 0/0을 0으로 바로 계산

**해결 핵심**
* ✓ 0/0 꼴은 부정형 → 인수분해나 유리화
* ✓ 공통인수를 약분하면 극한 계산 가능
* ✓ 분자, 분모가 모두 0으로 가는지 먼저 확인

**예시:** lim(x→2) (x² - 4)/(x - 2)의 값은?

---

#### 🔴 최상 ⭐⭐⭐ | ∞/∞ 부정형
**함정:** ∞/∞를 1로 계산하거나 처리 못함

**해결 핵심**
* ✓ 분자, 분모를 최고차항으로 나눔
* ✓ 분자 차수 > 분모 차수: 발산
* ✓ 분자 차수 = 분모 차수: 최고차 계수비
* ✓ 분자 차수 < 분모 차수: 0

**예시:** lim(x→∞) (3x² + 2x)/(x² - 1)의 값은?

---

### 함수의 연속 | 함정 2개

#### 🔴 최상 ⭐⭐⭐ | 연속의 세 조건
**함정:** 조건 하나만 확인하고 연속이라고 판단

**해결 핵심**
* ✓ ① f(a)가 정의됨
* ✓ ② lim(x→a) f(x)가 존재함
* ✓ ③ lim(x→a) f(x) = f(a)
* ✓ 세 조건 모두 만족해야 연속

**예시:** f(x) = (x² - 1)/(x - 1) (x ≠ 1), f(1) = 2일 때, x = 1에서 연속인가?

---

#### 🟠 상 ⭐⭐ | 미분가능 → 연속
**함정:** 연속이면 미분가능하다고 착각

**해결 핵심**
* ✓ 미분가능 → 연속 (항상 성립)
* ✓ 연속 → 미분가능 (성립 안 함)
* ✓ y = |x|는 x = 0에서 연속이지만 미분 불가능

**예시:** f(x) = |x|는 x = 0에서 미분가능한가?

---

### 미분계수와 도함수 | 함정 2개

#### 🔴 최상 ⭐⭐⭐ | 미분계수의 정의
**함정:** 극한에서 h → a로 착각

**해결 핵심**
* ✓ f'(a) = lim(h→0) [f(a+h) - f(a)]/h
* ✓ 또는 f'(a) = lim(x→a) [f(x) - f(a)]/(x-a)
* ✓ h는 0으로, x는 a로 감

**예시:** f(x) = x²의 x = 3에서의 미분계수를 정의를 이용해 구하시오.

---

#### 🟠 상 ⭐⭐⭐ | 상수의 미분
**함정:** 상수의 미분이 그 상수라고 착각

**해결 핵심**
* ✓ (상수)' = 0
* ✓ (cf(x))' = cf'(x) (상수 × 함수)
* ✓ (xⁿ)' = nxⁿ⁻¹

**예시:** (3x² + 5)'을 구하시오.

---

### 도함수의 활용 | 함정 2개

#### 🔴 최상 ⭐⭐⭐ | f'(a) = 0이면 극값?
**함정:** f'(a) = 0인 점이 항상 극값이라고 착각

**해결 핵심**
* ✓ f'(a) = 0이고 f'(x)의 부호가 바뀌어야 극값
* ✓ f'(x) = 0이어도 부호가 안 바뀌면 극값 아님
* ✓ 예: f(x) = x³, f'(0) = 0이지만 극값 아님

**예시:** f(x) = x³의 x = 0에서 극값을 가지는가?

---

#### 🔴 최상 ⭐⭐⭐ | 접선의 방정식
**함정:** 기울기를 f(a)로 착각

**해결 핵심**
* ✓ 점 (a, f(a))에서 접선의 기울기 = f'(a)
* ✓ 접선의 방정식: y - f(a) = f'(a)(x - a)
* ✓ 함숫값과 미분계수를 혼동하지 않기

**예시:** y = x²에서 x = 2인 점에서의 접선의 방정식은?

---

### 부정적분 | 함정 2개

#### 🔴 최상 ⭐⭐⭐ | 적분상수 누락
**함정:** + C를 빠뜨림

**해결 핵심**
* ✓ 부정적분은 반드시 + C
* ✓ 정적분은 C 없음 (소거됨)
* ✓ 초기 조건이 주어지면 C 결정

**예시:** ∫2x dx = x² + C (C 필수)

---

#### 🟠 상 ⭐⭐ | xⁿ의 적분
**함정:** ∫xⁿ dx = xⁿ⁺¹로 착각 (n+1로 안 나눔)

**해결 핵심**
* ✓ ∫xⁿ dx = xⁿ⁺¹/(n+1) + C (n ≠ -1)
* ✓ n = -1일 때: ∫(1/x) dx = ln|x| + C
* ✓ 미분하여 검산

**예시:** ∫x³ dx를 구하시오.

---

### 정적분 | 함정 2개

#### 🔴 최상 ⭐⭐⭐ | 정적분 계산 순서
**함정:** F(a) - F(b)로 계산 (순서 반대)

**해결 핵심**
* ✓ ∫[a to b] f(x) dx = F(b) - F(a)
* ✓ "위끝 - 아래끝"
* ✓ 적분 구간 바꾸면 부호 반대

**예시:** ∫[1 to 3] 2x dx의 값은?

---

#### 🟠 상 ⭐⭐⭐ | 기함수의 적분
**함정:** 기함수의 대칭 구간 적분이 0임을 모름

**해결 핵심**
* ✓ f(-x) = -f(x) (기함수)이면 ∫[-a to a] f(x) dx = 0
* ✓ f(-x) = f(x) (우함수)이면 ∫[-a to a] f(x) dx = 2∫[0 to a] f(x) dx
* ✓ 대칭성 활용하면 계산 단축

**예시:** ∫[-2 to 2] x³ dx의 값은?

---

### 정적분의 활용 | 함정 2개

#### 🔴 최상 ⭐⭐⭐ | x축 아래 부분의 넓이
**함정:** 음수 그대로 계산하면 넓이가 음수

**해결 핵심**
* ✓ 넓이는 항상 양수
* ✓ x축 아래 부분: 적분값에 -1을 곱하거나 |f(x)|를 적분
* ✓ 구간을 나누어 각각 양수화

**예시:** y = x² - 4와 x축 사이의 넓이를 구하시오. (-2 ≤ x ≤ 2)

---

#### 🔴 최상 ⭐⭐⭐ | 두 곡선 사이 넓이
**함정:** 위, 아래 곡선 구분 없이 계산

**해결 핵심**
* ✓ 넓이 = ∫[a to b] |f(x) - g(x)| dx
* ✓ f(x) ≥ g(x)이면 ∫[a to b] (f(x) - g(x)) dx
* ✓ 위아래가 바뀌는 구간에서 분리

**예시:** y = x²과 y = x + 2 사이의 넓이를 구하시오.

---

# 고등학교 (확률과 통계)

---

## 확률과 통계 (선택)

### 순열과 조합 | 함정 2개

#### 🔴 최상 ⭐⭐⭐ | 중복조합
**함정:** 중복순열과 중복조합 공식 혼동

**해결 핵심**
* ✓ 중복순열: nΠr = nʳ (순서 O, 중복 O)
* ✓ 중복조합: nHr = ₙ₊ᵣ₋₁Cᵣ (순서 X, 중복 O)
* ✓ 중복조합에서 n + r - 1에 주의

**예시:** 3종류의 과일 중에서 중복을 허용하여 5개를 선택하는 경우의 수는?

---

#### 🟠 상 ⭐⭐ | 조건부 순열
**함정:** 특정 조건(이웃, 이웃하지 않음)이 있는 순열

**해결 핵심**
* ✓ 이웃하는 경우: 묶어서 하나로 취급
* ✓ 이웃하지 않는 경우: 나머지 배열 후 빈자리에 배치
* ✓ 양 끝 조건: 양 끝 먼저 배열

**예시:** A, B, C, D, E를 일렬로 배열할 때, A와 B가 이웃하는 경우의 수는?

---

### 이항정리 | 함정 2개

#### 🔴 최상 ⭐⭐⭐ | 일반항의 r
**함정:** (a+b)ⁿ의 일반항에서 r을 1부터 시작

**해결 핵심**
* ✓ 일반항: ₙCᵣ aⁿ⁻ʳ bʳ, r은 0부터 n까지
* ✓ r+1번째 항의 r값 주의
* ✓ 특정 항 = x^k 형태 맞추기

**예시:** (x + 2)⁵의 전개식에서 x³의 계수는?

---

#### 🟠 상 ⭐⭐ | (-b)ʳ 부호
**함정:** (a-b)ⁿ에서 (-1)ʳ 처리 누락

**해결 핵심**
* ✓ (a-b)ⁿ의 일반항: ₙCᵣ aⁿ⁻ʳ (-b)ʳ = ₙCᵣ aⁿ⁻ʳ (-1)ʳ bʳ
* ✓ r이 홀수면 음수, 짝수면 양수
* ✓ 부호 결정 후 계수 계산

**예시:** (x - 1)⁶의 전개식에서 x⁴의 계수는?

---

### 확률 | 함정 2개

#### 🔴 최상 ⭐⭐⭐ | 덧셈정리에서 중복
**함정:** P(A∪B) = P(A) + P(B)로 계산 (중복 미제외)

**해결 핵심**
* ✓ P(A∪B) = P(A) + P(B) - P(A∩B)
* ✓ 배반사건이면 P(A∩B) = 0
* ✓ 벤 다이어그램으로 확인

**예시:** 주사위를 던져 3의 배수 또는 짝수가 나올 확률은?

---

#### 🟠 상 ⭐⭐⭐ | 여사건
**함정:** "적어도"를 직접 계산하다 경우 누락

**해결 핵심**
* ✓ P(적어도 하나) = 1 - P(하나도 아님)
* ✓ "적어도", "최소한" → 여사건 활용
* ✓ 직접 계산보다 훨씬 간단

**예시:** 동전 4개를 던질 때, 적어도 1개가 앞면일 확률은?

---

### 조건부확률 | 함정 2개

#### 🔴 최상 ⭐⭐⭐ | P(B|A)와 P(A|B) 혼동
**함정:** 조건의 위치를 바꿔서 계산

**해결 핵심**
* ✓ P(B|A) = P(A∩B)/P(A): A가 주어졌을 때 B
* ✓ P(A|B) = P(A∩B)/P(B): B가 주어졌을 때 A
* ✓ 조건(주어진 사건)이 분모에 들어감

**예시:** P(A) = 0.3, P(B) = 0.4, P(A∩B) = 0.1일 때, P(B|A)는?

---

#### 🔴 최상 ⭐⭐⭐ | 독립과 배반
**함정:** 독립사건과 배반사건을 같은 개념으로 착각

**해결 핵심**
* ✓ 독립: P(A∩B) = P(A)P(B) → 서로 영향 없음
* ✓ 배반: P(A∩B) = 0 → 동시에 일어나지 않음
* ✓ 독립 ≠ 배반 (완전히 다른 개념)

**예시:** P(A) = 0.3, P(B) = 0.5일 때, A, B가 독립이면 P(A∩B)는?

---

### 확률분포 | 함정 2개

#### 🔴 최상 ⭐⭐⭐ | 분산 공식
**함정:** V(X) = E(X)² - E(X²)로 착각

**해결 핵심**
* ✓ V(X) = E(X²) - {E(X)}²
* ✓ E(X²)가 먼저, {E(X)}²를 뺌
* ✓ 또는 V(X) = E((X - μ)²)

**예시:** E(X) = 3, E(X²) = 13일 때, V(X)는?

---

#### 🔴 최상 ⭐⭐⭐ | V(aX+b) 공식
**함정:** V(aX+b) = a²V(X) + b로 착각

**해결 핵심**
* ✓ E(aX+b) = aE(X) + b
* ✓ V(aX+b) = a²V(X) (b는 분산에 영향 없음)
* ✓ σ(aX+b) = |a|σ(X)

**예시:** V(X) = 4일 때, V(3X - 2)는?

---

### 정규분포 | 함정 2개

#### 🔴 최상 ⭐⭐⭐ | 표준화 공식
**함정:** Z = (X-μ)/σ에서 분자와 분모를 반대로

**해결 핵심**
* ✓ Z = (X - μ)/σ
* ✓ "값에서 평균을 빼고 표준편차로 나눈다"
* ✓ 표준화하면 N(0, 1)이 됨

**예시:** X ~ N(50, 10²)일 때, X = 70을 표준화하면?

---

#### 🟠 상 ⭐⭐⭐ | 정규분포표 대칭성
**함정:** P(Z ≤ -a) 계산에서 대칭성 미활용

**해결 핵심**
* ✓ P(Z ≤ -a) = P(Z ≥ a) = 1 - P(Z ≤ a)
* ✓ P(-a ≤ Z ≤ a) = 2P(0 ≤ Z ≤ a)
* ✓ 표준정규분포는 y축 대칭

**예시:** P(Z ≤ 1.5) = 0.9332일 때, P(Z ≥ -1.5)는?

---

### 통계적 추정 | 함정 2개

#### 🔴 최상 ⭐⭐⭐ | 신뢰구간 공식
**함정:** √n을 n으로 착각

**해결 핵심**
* ✓ 신뢰구간: x̄ ± z × σ/√n
* ✓ 분모는 √n (n이 아님!)
* ✓ n이 커지면 구간 폭이 좁아짐

**예시:** n = 100, σ = 10, x̄ = 50일 때, 95% 신뢰구간은?

---

#### 🟠 상 ⭐⭐ | n과 신뢰구간 폭
**함정:** n이 커지면 신뢰구간이 넓어진다고 착각

**해결 핵심**
* ✓ n ↑ → 신뢰구간 폭 ↓ (좁아짐)
* ✓ 신뢰도 ↑ → 신뢰구간 폭 ↑ (넓어짐)
* ✓ 표본 크기와 신뢰도는 반대 효과

**예시:** 표본 크기를 4배로 늘리면 신뢰구간 폭은?

---

# 고등학교 (미적분Ⅱ)

---

## 미적분Ⅱ (선택: 심화)

### 수열의 극한 | 함정 2개

#### 🔴 최상 ⭐⭐⭐ | 등비수열의 수렴
**함정:** |r| ≤ 1이면 수렴이라고 착각

**해결 핵심**
* ✓ |r| < 1: 수렴 (→ 0)
* ✓ r = 1: 상수 a₁으로 수렴
* ✓ r = -1: 진동 (발산)
* ✓ |r| > 1: 발산

**예시:** aₙ = (-1)ⁿ의 극한은?

---

#### 🟠 상 ⭐⭐ | lim aₙ = 0과 급수의 수렴
**함정:** lim aₙ = 0이면 Σaₙ이 수렴한다고 착각

**해결 핵심**
* ✓ lim aₙ ≠ 0이면 급수는 반드시 발산
* ✓ lim aₙ = 0이어도 급수는 수렴할 수도, 발산할 수도 있음
* ✓ 예: Σ(1/n)은 발산 (조화급수)

**예시:** lim(n→∞) 1/n = 0이지만 Σ(1/n)은?

---

### 급수 | 함정 2개

#### 🔴 최상 ⭐⭐⭐ | 등비급수 수렴 조건
**함정:** |r| ≥ 1일 때 합 공식 적용

**해결 핵심**
* ✓ |r| < 1일 때만 수렴, 합 = a/(1-r)
* ✓ |r| ≥ 1이면 발산 (합 없음)
* ✓ 수렴 조건 먼저 확인

**예시:** Σ(1/2)ⁿ (n=1 to ∞)의 합은?

---

#### 🟠 상 ⭐⭐ | 급수와 정적분
**함정:** 급수의 합을 정적분으로 바꾸는 연결 실패

**해결 핵심**
* ✓ lim(n→∞) (1/n) Σf(k/n) = ∫[0 to 1] f(x) dx
* ✓ 구분구적법의 역과정
* ✓ 급수를 리만합으로 해석

**예시:** lim(n→∞) (1/n)(1² + 2² + ... + n²)/n²의 값은?

---

### 여러 가지 함수의 미분 | 함정 2개

#### 🔴 최상 ⭐⭐⭐ | 지수/로그 미분 혼동
**함정:** (eˣ)' = xeˣ⁻¹로 착각

**해결 핵심**
* ✓ (eˣ)' = eˣ (자기 자신)
* ✓ (aˣ)' = aˣ ln a
* ✓ (ln x)' = 1/x
* ✓ (logₐ x)' = 1/(x ln a)

**예시:** (e²ˣ)'을 구하시오.

---

#### 🔴 최상 ⭐⭐⭐ | 삼각함수 미분 부호
**함정:** (cos x)' = sin x로 착각

**해결 핵심**
* ✓ (sin x)' = cos x
* ✓ (cos x)' = -sin x (마이너스!)
* ✓ (tan x)' = sec²x

**예시:** (sin x + cos x)'을 구하시오.

---

### 여러 가지 미분법 | 함정 2개

#### 🔴 최상 ⭐⭐⭐ | 합성함수 미분 (연쇄법칙)
**함정:** 속미분을 빠뜨림

**해결 핵심**
* ✓ {f(g(x))}' = f'(g(x)) × g'(x)
* ✓ 겉미분 × 속미분
* ✓ (sin 2x)' = cos 2x × 2 = 2cos 2x

**예시:** (e^(x²))'을 구하시오.

---

#### 🔴 최상 ⭐⭐⭐ | 몫의 미분 부호
**함정:** (f/g)' = (f'g + fg')/g²로 착각

**해결 핵심**
* ✓ (f/g)' = (f'g - fg')/g² (빼기!)
* ✓ 분자: 분자 미분 × 분모 - 분자 × 분모 미분
* ✓ 분모: 분모의 제곱

**예시:** (x/(x+1))'을 구하시오.

---

### 도함수의 활용 | 함정 2개

#### 🔴 최상 ⭐⭐⭐ | 이계도함수와 볼록성
**함정:** f''(x) > 0이면 위로 볼록이라고 착각

**해결 핵심**
* ✓ f''(x) > 0: 아래로 볼록 (∪)
* ✓ f''(x) < 0: 위로 볼록 (∩)
* ✓ 변곡점: f''(x) = 0이고 부호가 바뀌는 점

**예시:** f(x) = x³의 변곡점은?

---

#### 🟠 상 ⭐⭐ | 매개변수 미분
**함정:** dy/dx = (dx/dt)/(dy/dt)로 착각

**해결 핵심**
* ✓ dy/dx = (dy/dt)/(dx/dt)
* ✓ 분자가 y, 분모가 x
* ✓ 이계도함수: d²y/dx² = (d/dt(dy/dx))/(dx/dt)

**예시:** x = t², y = t³일 때, dy/dx는?

---

### 여러 가지 적분법 | 함정 2개

#### 🔴 최상 ⭐⭐⭐ | 치환적분 du 처리
**함정:** u = g(x)로 치환 후 du 처리 누락

**해결 핵심**
* ✓ u = g(x)이면 du = g'(x) dx
* ✓ dx = du/g'(x)로 바꾸기
* ✓ 정적분에서는 적분 구간도 바꿈

**예시:** ∫2x × e^(x²) dx를 치환적분으로 구하시오.

---

#### 🔴 최상 ⭐⭐⭐ | 부분적분 선택
**함정:** f와 g' 선택을 잘못하여 더 복잡해짐

**해결 핵심**
* ✓ LIATE 순서: 로그 > 역삼각 > 대수(다항) > 삼각 > 지수
* ✓ 앞에 있는 것을 f로, 뒤의 것을 g'로
* ✓ ∫x eˣ dx에서 f = x, g' = eˣ

**예시:** ∫x cos x dx를 부분적분으로 구하시오.

---

### 정적분의 활용 | 함정 2개

#### 🔴 최상 ⭐⭐⭐ | 회전체 부피 π 누락
**함정:** V = ∫{f(x)}² dx로 계산 (π 빠뜨림)

**해결 핵심**
* ✓ V = π∫[a to b] {f(x)}² dx
* ✓ 단면적 πr²을 적분
* ✓ y축 회전은 x와 y를 바꿔서

**예시:** y = x (0 ≤ x ≤ 1)을 x축 둘레로 회전한 회전체의 부피는?

---

#### 🔴 최상 ⭐⭐⭐ | 정적분으로 정의된 함수
**함정:** F(x) = ∫[a to g(x)] f(t) dt의 미분

**해결 핵심**
* ✓ F(x) = ∫[a to x] f(t) dt이면 F'(x) = f(x)
* ✓ 상한이 g(x)이면 F'(x) = f(g(x)) × g'(x)
* ✓ 합성함수 미분 적용

**예시:** F(x) = ∫[0 to x²] t dt일 때, F'(x)는?

# 고등학교 (기하)

---

## 기하 (선택)

### 이차곡선 (포물선, 타원, 쌍곡선) | 함정 3개

#### 🔴 최상 ⭐⭐⭐ | 포물선 초점과 준선
**함정:** y² = 4px에서 초점을 (4p, 0)으로 착각

**해결 핵심**
* ✓ y² = 4px: 초점 (p, 0), 준선 x = -p
* ✓ x² = 4py: 초점 (0, p), 준선 y = -p
* ✓ 4p에서 4로 나눈 값이 p

**예시:** y² = 8x의 초점과 준선은?

---

#### 🔴 최상 ⭐⭐⭐ | 타원과 쌍곡선의 c² 공식
**함정:** 타원과 쌍곡선에서 c² 공식 혼동

**해결 핵심**
* ✓ 타원: c² = a² - b² (빼기)
* ✓ 쌍곡선: c² = a² + b² (더하기)
* ✓ 타원은 a > c, 쌍곡선은 c > a

**예시:** 타원 x²/25 + y²/16 = 1의 초점의 좌표는?

---

#### 🟠 상 ⭐⭐ | 쌍곡선의 점근선
**함정:** 점근선을 y = ±(a/b)x로 착각

**해결 핵심**
* ✓ x²/a² - y²/b² = 1의 점근선: y = ±(b/a)x
* ✓ 분모에 a, 분자에 b
* ✓ 그래프는 점근선에 가까워짐

**예시:** x²/9 - y²/4 = 1의 점근선은?

---

### 벡터 | 함정 2개

#### 🔴 최상 ⭐⭐⭐ | 내적과 수직
**함정:** 내적 = 0이면 평행이라고 착각

**해결 핵심**
* ✓ a⃗·b⃗ = 0 ⟺ a⃗ ⊥ b⃗ (수직)
* ✓ 평행 조건: a⃗ = kb⃗ (실수배)
* ✓ 내적의 기하적 의미: |a⃗||b⃗|cos θ

**예시:** a⃗ = (2, 3), b⃗ = (3, -2)일 때, a⃗와 b⃗는 어떤 관계?

---

#### 🟠 상 ⭐⭐⭐ | 내적 성분 계산
**함정:** a⃗·b⃗ = a₁b₂ + a₂b₁로 착각

**해결 핵심**
* ✓ a⃗·b⃗ = a₁b₁ + a₂b₂ (같은 성분끼리)
* ✓ a⃗ = (a₁, a₂), b⃗ = (b₁, b₂)
* ✓ |a⃗|² = a⃗·a⃗ = a₁² + a₂²

**예시:** a⃗ = (1, 2), b⃗ = (3, 4)일 때, a⃗·b⃗는?

---

### 공간도형과 공간좌표 | 함정 2개

#### 🟠 상 ⭐⭐ | 공간 거리 공식
**함정:** z성분을 빠뜨림

**해결 핵심**
* ✓ d = √{(x₂-x₁)² + (y₂-y₁)² + (z₂-z₁)²}
* ✓ 평면 거리에 z성분 추가
* ✓ 구의 방정식에서도 세 성분 모두

**예시:** (1, 2, 3)과 (4, 6, 3) 사이의 거리는?

---

#### 🟠 상 ⭐⭐ | 좌표평면 위의 점
**함정:** xy평면 위의 점에서 z = 0인 것을 잊음

**해결 핵심**
* ✓ xy평면 위: z = 0
* ✓ yz평면 위: x = 0
* ✓ zx평면 위: y = 0

**예시:** 점 (3, 4, 5)에서 xy평면에 내린 수선의 발의 좌표는?
