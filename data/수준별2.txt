### 평면좌표

#### 🟥 하위권 전략
**목표:** 두 점 사이 거리와 내분점 공식 기초

**학습 포인트**
* ✓ 두 점 사이 거리 공식 √{(x₂-x₁)²+(y₂-y₁)²} 암기
* ✓ 내분점 공식: (mx₂+nx₁)/(m+n) 형태로 기억
* ✓ 중점은 1:1 내분점임을 이해

**추천 학습법:** 공식 암기 → 기본 계산 문제 20문제
**격려 멘트:** *"공식만 외우면 대입만 하면 돼요!"*

#### 🟨 중위권 전략
**목표:** 내분점, 외분점 구분과 무게중심

**학습 포인트**
* ✓ 내분점(더하기)과 외분점(빼기) 공식 구분
* ✓ 삼각형의 무게중심 = 세 좌표의 평균
* ✓ 도형의 넓이를 좌표로 구하기

**추천 학습법:** 내분/외분 혼합 문제 → 도형 연계
**격려 멘트:** *"내분은 더하고 외분은 빼요. 이것만 기억!"*

#### 🟩 상위권 전략
**목표:** 좌표를 이용한 도형 증명

**학습 포인트**
* ✓ 좌표를 설정하여 도형 성질 증명하기
* ✓ 점의 자취 문제
* ✓ 조건을 만족하는 점의 영역

**추천 학습법:** 증명 문제 → 자취 심화
**격려 멘트:** *"좌표로 증명하는 건 깔끔하고 논리적이에요!"*

### 직선의 방정식

#### 🟥 하위권 전략
**목표:** 기울기와 직선의 방정식 기초

**학습 포인트**
* ✓ 기울기 = (y증가량)/(x증가량) 확실히
* ✓ y - y₁ = m(x - x₁) 형태로 식 세우기
* ✓ 기울기 + 한 점, 두 점 → 직선의 방정식

**추천 학습법:** 직선의 방정식 세우기 반복
**격려 멘트:** *"기울기만 알면 직선은 끝이에요!"*

#### 🟨 중위권 전략
**목표:** 평행/수직 조건과 점과 직선 거리

**학습 포인트**
* ✓ 평행: 기울기 같음 (m₁ = m₂)
* ✓ 수직: 기울기 곱이 -1 (m₁ × m₂ = -1)
* ✓ 점과 직선 사이 거리 = |ax₁+by₁+c|/√(a²+b²)

**추천 학습법:** 위치 관계 → 거리 공식 집중
**격려 멘트:** *"거리 공식의 절댓값을 절대 잊지 마세요!"*

#### 🟩 상위권 전략
**목표:** 직선 심화와 영역

**학습 포인트**
* ✓ 두 직선의 교점을 지나는 직선의 방정식
* ✓ 직선과 좌표축으로 이루어진 도형의 넓이
* ✓ 부등식의 영역 기초

**추천 학습법:** 교점 활용 → 넓이 문제 → 영역
**격려 멘트:** *"직선은 미분에서 접선으로 다시 나와요!"*

### 원의 방정식

#### 🟥 하위권 전략
**목표:** 원의 방정식 표준형 이해

**학습 포인트**
* ✓ (x-a)² + (y-b)² = r²: 중심 (a,b), 반지름 r
* ✓ 괄호 안 부호와 반대가 중심 좌표임을 기억
* ✓ 일반형 → 표준형 변환 (완전제곱식)

**추천 학습법:** 중심, 반지름 읽기 → 변환 연습
**격려 멘트:** *"(x-3)이면 중심 x좌표는 +3이에요!"*

#### 🟨 중위권 전략
**목표:** 원과 직선의 위치 관계

**학습 포인트**
* ✓ 중심과 직선 사이 거리 d와 반지름 r 비교
* ✓ d < r: 두 점, d = r: 접함, d > r: 만나지 않음
* ✓ 현의 길이 구하기

**추천 학습법:** 위치 관계 판별 → 접선 조건
**격려 멘트:** *"거리와 반지름만 비교하면 돼요!"*

#### 🟩 상위권 전략
**목표:** 접선의 방정식과 두 원

**학습 포인트**
* ✓ 원 위의 점에서 접선, 원 밖의 점에서 접선
* ✓ 두 원의 위치 관계와 공통현
* ✓ 원과 직선이 만드는 도형의 넓이

**추천 학습법:** 접선 유형별 → 두 원 관계
**격려 멘트:** *"접선 문제는 기하의 꽃이에요!"*

### 도형의 이동

#### 🟥 하위권 전략
**목표:** 평행이동 기본 규칙

**학습 포인트**
* ✓ 점 (x,y) → (x+a, y+b)로 이동
* ✓ 방정식은 x 대신 x-a, y 대신 y-b 대입
* ✓ 점은 더하고, 식은 빼는 것 기억

**추천 학습법:** 점과 방정식 이동 각각 연습
**격려 멘트:** *"점은 +, 식은 -라고 외워요!"*

#### 🟨 중위권 전략
**목표:** 대칭이동 완벽 정리

**학습 포인트**
* ✓ x축 대칭: y → -y
* ✓ y축 대칭: x → -x
* ✓ 원점 대칭: x → -x, y → -y
* ✓ y = x 대칭: x ↔ y 교환

**추천 학습법:** 대칭 유형별 문제 정리
**격려 멘트:** *"각 대칭별로 뭐가 바뀌는지만 기억하세요!"*

#### 🟩 상위권 전략
**목표:** 복합 이동과 자취

**학습 포인트**
* ✓ 평행이동 + 대칭이동 합성
* ✓ 이동 후 도형의 방정식 구하기
* ✓ 점의 자취 문제

**추천 학습법:** 복합 이동 → 자취 문제
**격려 멘트:** *"이동을 순서대로 차근차근 적용하면 돼요!"*

**추천 학습법:** 그래프로 연속/불연속 판별

### 순열과 조합 (심화)

* ✓ 순열 ₙPᵣ = n!/(n-r)!
* ✓ 조합 ₙCᵣ = n!/r!(n-r)!
* ✓ 순서 있으면 순열, 없으면 조합

**추천 학습법:** 공식 → 기본 문제 반복

* ✓ 중복조합: ₙHᵣ = ₙ₊ᵣ₋₁Cᵣ
* ✓ 같은 것이 있는 순열

**추천 학습법:** 유형별 구분 → 복합 문제

**격려 멘트:** *"이항정리는 조합의 꽃이에요!"*

### 확률

#### 🟥 하위권 전략
**목표:** 확률의 기본 계산

**학습 포인트**
* ✓ 확률 = 해당 경우의 수 / 전체 경우의 수
* ✓ 0 ≤ P(A) ≤ 1
* ✓ 여사건: P(Aᶜ) = 1 - P(A)

**추천 학습법:** 기본 확률 계산 반복
**격려 멘트:** *"경우의 수를 잘 세면 확률은 쉬워요!"*

#### 🟨 중위권 전략
**목표:** 덧셈/곱셈 정리

**학습 포인트**
* ✓ P(A∪B) = P(A) + P(B) - P(A∩B)
* ✓ 독립이면 P(A∩B) = P(A)×P(B)
* ✓ "적어도" 문제 → 여사건 활용

**추천 학습법:** 덧셈/곱셈 정리 → 여사건
**격려 멘트:** *"적어도 문제는 1에서 빼세요!"*

#### 🟩 상위권 전략
**목표:** 확률 심화

**학습 포인트**
* ✓ 복잡한 확률 계산
* ✓ 확률의 곱셈정리 심화
* ✓ 다단계 확률 문제

**추천 학습법:** 복합 확률 → 다단계 문제
**격려 멘트:** *"확률은 경우를 나누어 생각하면 쉬워요!"*

### 조건부확률

#### 🟥 하위권 전략
**목표:** 조건부확률 정의

**학습 포인트**
* ✓ P(B|A) = P(A∩B)/P(A)
* ✓ "A가 주어졌을 때 B의 확률"
* ✓ 분모에 조건이 오는 것 기억

**추천 학습법:** 정의 → 기본 계산
**격려 멘트:** *"조건이 분모에 들어가요!"*

#### 🟨 중위권 전략
**목표:** 확률의 곱셈정리와 독립

**학습 포인트**
* ✓ P(A∩B) = P(A)×P(B|A)
* ✓ 독립: P(B|A) = P(B)
* ✓ 독립 ≠ 배반 (완전 다른 개념!)

**추천 학습법:** 곱셈정리 → 독립 판정
**격려 멘트:** *"독립과 배반은 다른 거예요!"*

#### 🟩 상위권 전략
**목표:** 조건부확률 심화

**학습 포인트**
* ✓ 베이즈 정리
* ✓ 원인의 확률
* ✓ 복잡한 조건부확률

**추천 학습법:** 베이즈 → 역방향 확률
**격려 멘트:** *"베이즈 정리는 원인을 추론하는 거예요!"*

### 확률분포

#### 🟥 하위권 전략
**목표:** 확률변수와 기댓값

**학습 포인트**
* ✓ 이산확률변수의 정의
* ✓ 확률분포표 작성
* ✓ E(X) = Σxᵢpᵢ (기댓값)

**추천 학습법:** 분포표 → 기댓값 계산
**격려 멘트:** *"기댓값은 평균이에요!"*

#### 🟨 중위권 전략
**목표:** 분산과 표준편차

**학습 포인트**
* ✓ V(X) = E(X²) - {E(X)}²
* ✓ σ(X) = √V(X)
* ✓ E(aX+b) = aE(X)+b, V(aX+b) = a²V(X)

**추천 학습법:** 분산 공식 → 성질 활용
**격려 멘트:** *"E는 그대로, V는 제곱!"*

#### 🟩 상위권 전략
**목표:** 이항분포

**학습 포인트**
* ✓ B(n,p)의 확률 공식
* ✓ E(X) = np, V(X) = np(1-p)
* ✓ 이항분포 활용 문제

**추천 학습법:** 이항분포 → 정규분포 연결
**격려 멘트:** *"이항분포는 확률분포의 대표 선수!"*

### 통계적 추정

#### 🟥 하위권 전략
**목표:** 모집단과 표본

**학습 포인트**
* ✓ 모집단, 표본, 모평균, 표본평균 구분

**추천 학습법:** 용어 정리 → 기본 개념
**격려 멘트:** *"모집단은 전체, 표본은 일부예요!"*

#### 🟨 중위권 전략
**목표:** 표본평균의 분포

**학습 포인트**

* ✓ 표본평균의 정규분포
* ✓ 중심극한정리

**추천 학습법:** 분포 → 중심극한정리
**격려 멘트:** *"n이 커지면 정규분포에 가까워져요!"*

#### 🟩 상위권 전략
**목표:** 모평균 추정과 신뢰구간

**학습 포인트**
* ✓ 신뢰구간: x̄ ± z×σ/√n
* ✓ 95% → z=1.96, 99% → z=2.58
* ✓ n이 커지면 신뢰구간이 좁아짐

**추천 학습법:** 신뢰구간 → 응용 문제
**격려 멘트:** *"신뢰구간은 '믿을 수 있는 범위'예요!"*
